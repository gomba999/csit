Why use SLIM?

SLIM is the messaging layer for agent systems: named endpoints, multiple interaction patterns, optional routing across nodes, and a control plane for cross-node links.

This page is CSIT’s repeatable evidence for those capabilities. Performance numbers are in the appendix only.
